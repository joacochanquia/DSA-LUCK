Información sobre SQL Injection

La inyección SQL es una técnica de ataque que permite a un atacante interferir con las consultas que una aplicación hace a su base de datos.

Este tipo de vulnerabilidad puede permitir a un atacante ver datos que normalmente no podría recuperar, como datos de otras cuentas.

En esta página puedes aprender sobre la vulnerabilidad. Para probar la base de datos vulnerable, haz clic en el botón de abajo.
